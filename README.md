This is a Vite-powered React project converted from a Next.js starter.

## Getting Started

Install dependencies if you have not already:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `src/App.tsx`. The page will auto-update as you edit the file.

## Build

Build the production bundle with:

```bash
npm run build
```

Preview the production build with:

```bash
npm run preview
```

## Notes

- The contact form and backend API routes have been removed.
- The site is now a frontend-only React app.
- Tailwind CSS is still used for styling.
